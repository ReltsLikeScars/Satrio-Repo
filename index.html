bold dan italic

bold (b)
adalah menebalkan suatu kata atau kalimat.
contoh : saya hacker. menjadi : saya hacker

italic (i)
adalah membuat miring suatu kata atau kalimat.
contoh : saya ingin menjadi bug hunter.menjadi : saya ingin menjadi bug hunter

underline,superscript,subscript

underline (u)
adalah memberi garis di bawah kata.
contoh : otw jepang. menjadi : otw jepang.

superscript (sup)
adalah membuat tulisan atau angka menjadi kecil di bagian atas.
contoh : E=MC2. menjadi E=MC2

subscript (sub)
adalah membuat tulisan atau angka menjadi kecil di bagian bawah.
contoh : H2O. menjadi H2O

line break dan horizontal rule

line break (br)
adalah memberi jeda tanpa spasi untuk kata.
contoh : oksigen adalah suatu zat. menjadi : oksigen
adalah suatu zat

horizontal rule (hr)
adalah memberi garis mendatar di bawah/di atas kata.
contoh : bab tentang sholat. menjadi :

bab tentang sholat

strong,emphasis,blockquote dan quote

strong (strong)
adalah untuk menebalkan suatu kalimat atau kata. walaupun fungsinya sama seperti tag bold tetapi tag strong lebih tepat untuk menunjukkan pentingnya suatu teks.
contoh : saya ingin cerdas. menjadi saya ingin cerdas

emphasis (em)
adalah untuk memiringkan suatu kata atau kalimat,walaupun fungsinya sama dengan tag italic tetapi tag emphasis lebih cocok untuk menunjukkan penekanan semantik.
contoh : saya ingin pintar. menjadi : saya ingin pintar

blockquote (blockquote)
adalah menjadikan kata atau kalimat menjadi terberi sedikit spasi dari samping. contoh : satrio. menjadi :

satrio

quote (q)
adalah memberi suatu tanda petik pada kata atau kalimat. contoh : kata ibu saya, saya harus selalu menjaga kesehatan. menjadi : kata ibu saya saya harus selalu menjaga kesehatan.
